## Visual Design Improvements

### Color & Branding
- Consistent use of primary red (#E31E24)
- Dark/light section alternation for visual rhythm
- Gradient backgrounds for depth
- Glassmorphism effects (backdrop-blur)

### Animations
- Framer Motion scroll animations
- Hover effects on all interactive elements
- Scale, rotate, and translate transforms
- Pulsing/breathing animations for CTAs

### Layout Enhancements
- Responsive grid layouts
- Proper spacing and breathing room
- Visual hierarchy with typography
- Background decorative elements (blurred circles)

### Call-to-Actions
- Multiple CTAs throughout the page
- Phone numbers are prominent and clickable
- data-cta attributes for tracking
- Variety of CTA styles (primary, secondary, outline)
